# SSMEAS

Smart Sewage Monitoring and Early Alert System: ESP32 telemetry through ThingSpeak, a secured Express/PostgreSQL API, and a live Next.js operations dashboard.

Start with [Installation Guide](docs/Installation-Guide.md), then use the [Acceptance Test Report](docs/Acceptance-Test-Report.md) and [Demonstration Script](docs/Demonstration-Script.md). Architecture and requirements are documented under `docs/`.
